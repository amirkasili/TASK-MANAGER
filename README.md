# 📝 Task Manager CLI

A simple and efficient **Task Manager** you can run in your terminal.  
Built with a focus on **SOLID principles**, this project ensures clean, maintainable, and extensible code.  

---

## 🚀 Features
- ➕ Add new tasks  
- ✏️ Edit existing tasks  
- ❌ Delete tasks  
- 🔍 Filter tasks (e.g., by status or keyword)  
- 🖥️ Command-Line Interface (CLI) for smooth usage
